# ROI History

This folder contains monthly aggregates of ROI tracking data.

## File Format

Files are named `[YYYY-MM].md` (e.g., `2026-01.md`)

## Structure

Each monthly file contains:
- Daily session summaries
- Monthly totals
- Top skills by time saved
- Skill distribution

## Usage

This data powers the `/roi-report` skill for 30/90 day summaries.

## Maintenance

- Files are created automatically when first session of month completes
- Historical files can be archived annually
