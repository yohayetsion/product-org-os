# Product Organization RACI Matrix

## RACI Definition

| Letter | Meaning | Description |
|--------|---------|-------------|
| **R** | Responsible | Does the work |
| **A** | Accountable | Makes final decision, owns outcome |
| **C** | Consulted | Provides input before decision |
| **I** | Informed | Notified after decision |

**Rule**: Every activity has exactly ONE Accountable owner.

## V2V Phase Mapping

The RACI matrix maps to the V2V Operating System phases:

| V2V Phase | Primary Activities | Key Accountability |
|-----------|-------------------|-------------------|
| **Phase 1: Strategic Foundation** | Market & Customer Intimacy | CPO/VP Product (A) |
| **Phase 2: Strategic Decisions** | Pricing Strategy, Business Plan | VP Product (A), BizOps (A) |
| **Phase 3: Strategic Commitments** | Product Vision & Roadmap, GTM | VP Product (A), Dir PMM (A) |
| **Phase 4: Coordinated Execution** | Product Delivery Planning, Requirements | Dir PM (A), PM (R) |
| **Phase 5: Business Outcomes** | Value Realization | Value Realization (R) |
| **Phase 6: Learning Loop** | Organizational Processes | Product Ops (R) |

## How to Use This Matrix

### When Starting Work
1. Identify the activity type
2. Find who is Accountable (A)
3. Confirm with the A before proceeding
4. Identify who to Consult (C)
5. Plan who to Inform (I)

### When Making Decisions
1. A makes the final call
2. R does the work to get there
3. C provides input before
4. I is notified after

### When Things Are Unclear
1. Default to escalating to A
2. If no A is clear, clarify before proceeding
3. Document the RACI for future reference
